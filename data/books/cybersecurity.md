# Cybersecurity Basics

Cybersecurity involves protecting systems, networks, and data from digital attacks.

## Common Threats

- Phishing
- Malware
- Ransomware
- Man-in-the-middle attacks

## Phishing

Phishing is a social engineering attack where attackers trick users into revealing sensitive information.

## Security Practices

- Use strong passwords
- Enable two-factor authentication (2FA)
- Avoid suspicious links
- Keep software updated

## Tools

- Firewalls
- Antivirus software
- Intrusion detection systems

## CIA Triad

- Confidentiality → Protect data privacy
- Integrity → Ensure data accuracy
- Availability → Ensure system access
