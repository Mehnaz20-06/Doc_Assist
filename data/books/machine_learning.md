# Machine Learning

Machine Learning (ML) is a branch of Artificial Intelligence (AI) that enables systems to learn patterns from data and make decisions or predictions without being explicitly programmed.

Instead of writing rules manually, we provide data, and the model learns from it.

---

## How Machine Learning Works

1. **Data Collection** – Gather relevant data.
2. **Data Preprocessing** – Clean and format the data.
3. **Model Training** – Use algorithms to learn patterns.
4. **Evaluation** – Check how well the model performs.
5. **Prediction/Deployment** – Use the model on new data.

---

## Types of Machine Learning

### 1. Supervised Learning
- The model learns from **labeled data** (input + correct output).
- Goal: Predict outputs for new inputs.

**Examples:**
- Predicting house prices
- Email spam detection

**Common Algorithms:**
- Linear Regression
- Logistic Regression
- Decision Trees
- Support Vector Machines (SVM)

---

### 2. Unsupervised Learning
- The model works with **unlabeled data**.
- Goal: Find hidden patterns or groupings.

**Examples:**
- Customer segmentation
- Anomaly detection

**Common Algorithms:**
- K-Means Clustering
- Hierarchical Clustering
- PCA (Dimensionality Reduction)

---

### 3. Reinforcement Learning
- The model learns by **interacting with an environment**.
- It receives **rewards or penalties** based on actions.

**Examples:**
- Game-playing AI (chess, video games)
- Robot navigation

**Key Concepts:**
- Agent
- Environment
- Reward
- Policy

---

## Key Concepts in Machine Learning

- **Features**: Input variables used for prediction
- **Labels**: Output or target variable
- **Model**: Mathematical representation of patterns
- **Training**: Learning from data
- **Overfitting**: Model memorizes data, performs poorly on new data
- **Underfitting**: Model fails to learn patterns properly

---

## Applications of Machine Learning

- Recommendation systems (Netflix, Amazon)
- Image and speech recognition
- Fraud detection
- Medical diagnosis
- Autonomous vehicles

---

## Advantages

- Automates decision-making
- Improves with more data
- Handles complex patterns

---

## Limitations

- Requires large datasets
- Can be biased if data is biased
- Needs computational power

---

## Summary

Machine Learning allows computers to learn from data and improve over time without explicit programming. It is widely used across industries and is a core part of modern AI systems.
